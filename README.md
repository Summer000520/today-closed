# 今日不营业

一个只收压力、不收钱的解压网站原型。

## 已包含功能

- 烦恼回收站：输入烦恼后选择粉碎、放飞或沉入海底。
- 压力雾化器：点击生成虚拟雾气效果，不涉及真实吸烟。
- 幻想商店：假装购买虚拟商品，不涉及真实消费。
- 呼吸圆：用动画引导吸气、停留、呼气。
- 白噪音：用浏览器生成雨声、海浪、微风类底噪。

## 打开方式

直接打开 `index.html` 即可预览。

## 后续适合用 Vue 改造的方向

1. 把五个解压工具拆成 Vue 组件。
2. 用 `ref` 和 `reactive` 管理输入内容、当前页面、音量和购买记录。
3. 用 `v-for` 渲染幻想商店商品列表。
4. 用 `v-if` 或动态 class 切换不同解压面板。
5. 后续再加入路由、主题切换、动画库和本地心情记录。
