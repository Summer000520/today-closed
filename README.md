# 今日不营业

一个只收压力、不收钱的治愈系解压网站原型。当前版本是「今日不营业 2.1 大舞台版」，包含多房间入口、心情值统计和多个虚拟解压玩法。

## 已包含功能

- 今日小屋：作为多房间入口，按心情推荐解压工具。
- 烦恼回收站：输入烦恼后选择粉碎、纸飞机、沉入海底或变成星屑。
- 压力雾化器：点击生成虚拟雾气效果，不涉及真实吸烟。
- 幻想商店：假装购买虚拟商品，支持购物车和假装付款。
- 泡泡纸：点击泡泡获得轻量反馈。
- 桌面整理：点击散落物件，让桌面慢慢变清爽。
- 情绪漂流瓶：写一句话，让它离开屏幕。
- 呼吸圆：用动画引导吸气、停留、呼气。
- 白噪音：用浏览器生成雨声、海浪、微风和低频底噪。

## 打开方式

直接打开 `index.html` 即可预览。

## 后续适合用 Vue 改造的方向

1. 把顶部导航、今日小屋和各个解压工具拆成 Vue 组件。
2. 用 `ref` 和 `reactive` 管理输入内容、当前页面、音量和购买记录。
3. 用 `v-for` 渲染幻想商店商品列表。
4. 用 `v-if` 或动态 class 切换不同解压面板。
5. 后续再加入路由、主题切换、动画库和本地心情记录。
